# 언러닝 시작 가이드 (새 학생용 전체 안내)

첫 접속하셨나요? 이 페이지가 출발점입니다.
아래를 순서대로 따라가면 15분 안에 EarnLearning의 핵심 기능을 모두 익혀요.

## EarnLearning 이 제공하는 것

"스타트업을 위한 코딩입문" 수업을 시뮬레이션하는 게임화 플랫폼. 실제 창업가처럼 회사를 만들고, 외주를 주고받고, 투자를 받고, 주주총회를 열고, 필요하면 청산까지.

## Step 1. 가입 + 승인 대기

`/register` 에서 이메일·비밀번호·이름·학번 입력. 제출 후 관리자(교수/조교) **승인**을 기다립니다 — 승인되기 전에는 피드 접근 불가.

## Step 2. 반 입장

승인 후 로그인 → 교수님이 공유한 **반 코드** 를 입력 → 반 멤버가 되면 피드/지갑/마켓 전체 접근 가능.

## Step 3. 홈 화면 둘러보기

하단에 **5개 탭**:

| 탭 | 역할 |
|---|---|
| 홈 | 피드/타임라인 |
| 자산 | 개인 지갑 |
| 마켓 | 프리랜스 구인/지원 |
| 회사 | 내 회사 경영 |
| 더보기 | 투자/그랜트/대출/상점 등 |

## 꼭 알아야 할 개념

### 개인 지갑 ≠ 법인 지갑
개인(나)과 법인(회사)은 **별개 인격체** 입니다. 회사 돈을 개인 마음대로 가져가면 **횡령**.

### 주식 · 지분율
회사를 설립하면 총 10,000주가 발행되고 대표가 100% 소유. 투자 라운드로 주식을 양도하면 제3자(투자자)가 주주가 됩니다. 의사결정 때 그 사람들도 투표권을 가져요.

### 청산 시 세금 20%
회사를 청산하면 자산의 20%는 세금, 80%만 주주에게 분배됩니다. 무분별한 청산은 큰 비용을 동반하니 전략적으로 판단하세요.

## 프로필 · 채널

`/profile` 에서 이름/아바타/소개·학과·정보를 관리. DM 도 가능 (피드에서 사람 이름 클릭 → 메시지 보내기).

## FAQ

**Q. 승인 대기 줄 모르게 하려면?**
교수님/조교가 알림 받을 수 있도록 신청 직후 반 단톡방에 학번·이름을 알려드리세요. 관리자 페이지에 대기자 목록이 있어요.

**Q. 내 돈이 날아갈 수 있나요?**
개인 지갑은 오직 승인된 거래/청산/배당 규칙으로만 변동합니다. 시스템이 임의로 빼가는 일은 없어요. 단, 투자 실패·청산 세금·외주 미정산 등은 주주 본인이 감수하는 리스크입니다.
