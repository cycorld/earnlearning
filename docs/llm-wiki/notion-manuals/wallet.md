# 개인 지갑 완전 가이드

## 내 지갑이란?

학생 각자가 가지고 있는 **개인 계좌**입니다. 초기 승인 시 지급되는 종잣돈(반별 설정)으로 시작해서, 외주·투자·대출·청산 분배 등으로 잔액이 바뀝니다.

## 송금하기

우측 `송금` 버튼 → 받는 사람 선택 → 금액 입력. 개인 간 송금뿐 아니라 **법인(회사) 지갑으로도 보낼 수 있어요**.

**잠깐!** 법인 계좌에서 대표 개인 계좌로 마음대로 빼가는 건 현실에선 **횡령**입니다. 법인 자산의 "공적 성격"을 감안해 꼭 필요한 목적으로만 이동하세요.

## 거래 내역

지갑 하단 `전체 내역 보기` 또는 `/wallet/transactions` 로 이동하면 모든 입출금이 시간순으로 나옵니다.

각 거래는 `tx_type` 으로 분류됩니다:
- `transfer` — 개인↔개인/법인 송금
- `loan_grant` / `loan_repay` — 대출 / 상환
- `liquidation_payout` — 청산 분배금
- `engagement_reward` — 활동 리워드
- `freelance_pay` — 외주 정산
- `llm_billing` — LLM API 사용료 일일 차감 (매일 03:33 KST)

## FAQ

**Q. 잔액이 마이너스가 될 수 있나요?**
개인 지갑은 원칙적으로 마이너스 불가 (대출 상환 실패, LLM 사용료 부채 같은 일부 예외 제외). 송금 시 잔액 부족하면 거부됩니다.

**Q. 친구가 돈을 빌려달라는데 바로 보내면 되나요?**
개인 간 송금은 "증여"처럼 처리되어 기록만 남고 별도 계약이 없습니다. 나중에 돌려받으려면 계약(메시지/글)으로 증거를 남겨두세요. 공식 대출은 은행(`/bank`) 기능을 쓰는 게 안전합니다.
