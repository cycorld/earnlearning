# 투자 라운드 완전 가이드 (대표용 + 투자자용)

## 한 장 요약

- **투자 라운드 = 회사가 새 주식을 발행해서 자본을 모으는 이벤트**
- 대표는 "얼마 모을지(목표 금액)" + "몇 % 지분 줄지(제공 지분)" 만 정함
- 투자자는 원하는 만큼 주식 수를 입력해 매수 (분할 투자 가능)
- 라운드 결과: ① 100% 모집 → 자동 마감 / ② 부분 모집 + 조기 마감 / ③ 취소(환불) / ④ 만료(`failed`)

## 투자 도메인 기초

| 용어 | 뜻 | 예시 |
|---|---|---|
| 목표 금액 (Target) | 이번 라운드로 모을 총 자금 | 1,000,000원 |
| 제공 지분 (Offered %) | 투자자에게 넘길 지분율 | 20% |
| 포스트머니 가치 | 목표 ÷ 제공 지분 | 5,000,000원 |
| 프리머니 가치 | 포스트머니 − 목표 | 4,000,000원 |

### 주당 가격 계산

```
새 주식 수 = 기존 주식 × 제공 지분 ÷ (1 − 제공 지분)
주당 가격 = 목표 금액 ÷ 새 주식 수
```

예: 10,000주 회사가 20% 라운드 열면 새 주식 2,500주, 주당 400원. (시스템 자동 계산)

## 라운드 여는 법 (대표용)

1. `회사` 탭 → 본인 회사 → 스크롤해서 `투자 유치` → `+ 라운드 개설`
2. 목표 금액 + 제공 지분(1~99%) 입력. 보통 5~25% 적당.
3. 실시간 가치평가 미리보기 확인. **다운라운드**(프리머니 < 현재 기업가치) 경고가 뜨면 신중하게.
4. 개설 → 투자라운지 채널에 자동 포스트.

**동시에 한 라운드만** 가능. 진행 중이면 새 라운드 불가.

## 투자하는 법 (투자자용)

1. `/invest` → `투자 라운드` 탭
2. 카드에 모집 진행률 바 → 클릭해서 상세 페이지
3. 매수 주식 수 입력 → 실시간 결과 미리보기 (매수 금액, 취득 지분, 투자 후 총 주식)
4. 매수 → 내 지갑에서 차감 + 회사 법인 계좌 입금 + 나는 주주가 됨
5. 마지막 한 주 매수 시 "라운드 마감 + 회사 가치 재평가" 알림

## 시나리오별 처리

### 시나리오 A — 100% 모집 완료
- 상태: `open` → `funded` 자동 전환
- 회사 가치 = 포스트머니로 재평가
- 모든 주주에게 "라운드 마감" 알림

### 시나리오 B — 부분 모집 + 대표가 조기 마감
라운드 상세에 `대표자 도구` 카드 표시 (대표 본인에게만). "조기 마감":
- 상태: `open` → `funded` (실제 모금액으로 확정)
- 남은 주식은 발행 안 됨 (희석 덜 됨)
- 회사 가치 = 주당 가격 × 새 총 주식 수로 재평가

### 시나리오 C — 대표가 라운드 취소 (전액 환불)
- 모든 투자자에게 본인이 낸 금액 그대로 환불
- 발행됐던 주식 회수, `total_shares`/`total_capital` 원복
- 회사 가치는 건드리지 않음
- 상태: `open` → `cancelled`

**안전 가드**: 회사 지갑 잔액 부족 시 취소 불가 (이미 돈을 썼다면). 현실적 교훈: 모금한 돈을 막 쓰면 되돌리기 어렵다.

### 시나리오 D — 라운드 만료 (`failed`)
`expires_at` 지나면 자동으로 `failed`. 자동 환불은 안 들어감. 부분 매수 투자자는 그대로 주주로 남음.

## 포트폴리오 & 배당

`/invest` `내 포트폴리오` 탭에서 각 회사별:
- 보유 주식(수량 + 지분율)
- 투자금 (총 누적)
- 현재 가치 = 회사 가치 × 내 지분율
- 수익/손실 = 현재 가치 − 투자금
- 받은 배당금 (누적)

**배당금**: 대표가 "X원 배당" 결정 시 자동으로 **지분율 비례** 분배. 예) 100만원 배당 + 내 지분 20% → 20만원 입금.

## FAQ

**Q. 본인 회사에 본인이 투자할 수 있나요?**
아니요. `자신의 회사에 투자할 수 없습니다` 에러.

**Q. 한 라운드를 여러 명이 나눠서 투자할 수 있나요?**
네. 분할 투자 지원. 마지막 한 주가 팔리면 자동 마감.

**Q. 매수 주식 수를 더 입력하면 어떻게 되나요?**
남은 주식 초과 시 `남은 주식 수를 초과했습니다` 에러.

**Q. 동시에 여러 라운드를 열 수 있나요?**
아니요. 진행 중 라운드 있으면 새 라운드 불가.

**Q. 배당받은 후 회사가 청산되면?**
배당금은 그대로 내 지갑. 청산은 "남은 회사 자산" 분배하는 별도 이벤트.

## 알림 종류

- `investment_received` — 투자자 본인, 매수 직후
- `investment_funded` — 회사 대표, 투자 발생 / 마감 / 취소 시
- `dividend_received` — 주주, 배당 입금 시
