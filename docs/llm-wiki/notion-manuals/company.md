# 회사 설립·경영 완전 가이드

## 회사란?

학생 창업가가 만드는 **법인**입니다. 개인과 별개의 인격체로, 고유 지갑/주식/공시/지분 구조를 갖습니다. 주주총회로 의사결정하고, 외주·투자·대출·청산으로 회사를 변화시킵니다.

## 회사 설립

`/company/new` 에서 로고·회사명·소개·초기자본금 입력. **설립 시 내 개인 지갑에서 초기자본금이 법인 지갑으로 이전**됩니다.

설립 직후:
- `owner_id` 도 본인 + **전체 주식(10,000주)의 100%를 소유** (founding)
- `business_card` 는 빈 JSON 으로 시작
- `status` 는 `active`

## 회사 상세 페이지

회사 기본 정보 + 주요 섹션(공시/주주총회/투자 유치/법인 계좌 관리) 이 한 눈에.

**대표의 권한**: 회사 정보 수정(연필 버튼), 공시 작성, 안건 상정, 투자 라운드 개설, 청산 실행(가결 후).
**주주의 권한**: 투표(찬성/반대), 의견 작성(댓글/메시지), 배당 수령.

## 법인 계좌

회사 상세 하단 `법인 계좌 관리` → 전용 페이지 (잔액·거래 내역·송금).

## 명함 페이지

회사 대외 프로필. 서비스 URL, SNS, 태그라인 — "그 회사가 무엇을 하는지" 한 장에 알리는 페이지.

## FAQ

**Q. 회사를 몇 개까지 가질 수 있나요?**
개수 제한은 없지만, 각 회사의 초기자본금을 매번 넣어야 해서 자본이 있는 만큼만 가능. 권장은 **한 번에 1개에 집중**.

**Q. 회사 이름을 바꿀 수 있나요?**
네, 상세 페이지 우측 연필 아이콘에서 수정 가능 (대표만).
