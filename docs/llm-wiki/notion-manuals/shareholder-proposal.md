# 주주총회(일반 안건) 완전 가이드

## 주주총회란?

회사의 **다수결**을 통해 의사결정하는 공식 절차. 대표 맘대로 결정하지 않고, 주주들의 **지분율 기반 투표**로 중요 결정을 내립니다.

## 두 가지 안건 종류

| 종류 | 기본 가결 기준 | 언제 쓰나요? |
|---|---|---|
| 일반 안건 | 찬성 50% | 배당 승인, 사업 방향 합의, 주요 계약 승인 등 |
| 회사 청산 | 찬성 70% | 회사 문 닫기 (영구 정지). 가결 시 자산 자동 분배 |

## 안건 상정

회사 상세 페이지 → 주주총회 섹션 → **\[+ 안건 상정]** 버튼. 주주면 누구나 상정 가능 (대표 독점 X).

입력 항목:
- **제목** (필수)
- **설명**: 마크다운 허용. 근거·예상 효과·대안 등을 명확히
- **가결 기준(%)**: 기본 50% (1~100 범위)
- **투표 기간(일)**: 기본 7일

## 투표

주주의 **지분율이 투표 가중치**. 100주 주주와 1주 주주의 한 표 무게가 다릅니다. 찬성/반대 지분율 합산이 기준을 넘으면 즉시 가결, 기간 내 못 넘으면 부결.

**우리 시스템의 다수결 법칙**: "인원수"가 아닌 **찬성 주식 수 ÷ 전체 주식 수**. 대주주가 상대적으로 강한 권한을 가지는 실제 주식회사와 동일.

## 결과 알림

안건 종료 시 주주 전체에게 알림 (`주주총회 결과`). 투표용 파이 차트와 함께 기록이 남음.

청산 안건은 **가결 즉시 자동 집행**. 이전과 달리 대표가 별도 버튼을 누를 필요 없음.

## FAQ

**Q. 본인이 상정한 안건에 자기가 투표할 수 있나요?**
네, 가능합니다. 실제 주주총회에서도 제안자가 찬성하는 것이 자연스럽죠.

**Q. 안건 통과 후 실제 집행은 누가 하나요?**
일반 안건은 **선언적** 성격이 강함 (예: "신규 사업진출에 찬성" = 방향성 합의). 구체적 집행 행위는 대표가 별도로 합니다.

**Q. 동일 종류 안건을 동시에 여러 개 올릴 수 있나요?**
안됩니다. 동시 진행 안건은 각 종류 당 1개까지만.
