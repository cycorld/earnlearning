## 프롬프트 원문

db 는 sqlite으로만 사용해줘. 배포시 도커 볼륨을 설정해서 persistant로 진행해줘. redis 는 필요없어. oauth 는 스킵하고 학생들이 학교 이메일로 편하게 가입하도록 해줘. 대신 내가 승인한 학생만 접근가능해. 승인 안하면 승인 대기 안내문을 보여주면 돼. cyc@snu.ac.kr 로 문의 표시. 학생 정보는 이메일, 이름, 학과, 학번(full) 받으면 돼. 학번은 학생들끼리 보일 때는 년도(앞 2자리)만 보이도록 하고 최종 어드민은 전체 학번을 볼 수 있어야해.
