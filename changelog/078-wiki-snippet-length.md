# 078. search_wiki 스니펫 길이 12→64 (잘려 나가던 핵심 정보 복구)

**날짜**: 2026-04-19
**태그**: 챗봇, RAG, FTS5, 버그수정

## 증상
"주주총회 가결 기준?" 질문에 LLM 이 "문서에 명시되지 않았습니다" 로 답변. 하지만
`shareholder-proposal.md` 에는 **"일반 안건 50%, 회사 청산 70%"** 가 명시돼 있음.

## 원인
FTS5 `snippet()` 의 n_tokens 인자를 **12 토큰**으로 설정했던 것. 질문어 주변 12 토큰만
반환하다 보니 실제 가결 비율이 들어 있는 표 부분이 전부 `…` 로 잘려 나감:

```
snippet(chat_wiki_docs, 2, '[', ']', ' … ', 12)
                                             ^^ 너무 작음
```

결과 예시:
```
## 주주총회(일반 안건) 완전 가이드 …
 … 가결 시 자산 자동 분배 |  ← 핵심 수치 없음
```

## 수정
`snippet(..., 64)` 로 증가. 답변의 실제 근거(표 내용, 정책 문장)가 충분히 포함될 정도.

## 기대 효과
LMS 정책 관련 질문이 "문서에 없다" 대신 실제 문서 내용을 인용한 답으로 개선.
