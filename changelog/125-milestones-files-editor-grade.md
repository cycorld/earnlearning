# 125. 평가지표 페이지 — 사업계획서 비공개 첨부 + 회고 에디터 + 성적/자산 percentile

**날짜**: 2026-06-12
**티켓**: #125
**타입**: 신규 기능

## 무엇을 했나요?

평가지표(/milestones) 페이지를 세 가지로 업데이트했습니다.

1. **사업계획서 비공개 첨부** — 다양한 포맷(PDF·DOCX·PPTX·HWP 등) 다중 파일 업로드. **업로더 본인과 교수(관리자)만** 열람·다운로드 가능.
2. **회고 발표 에디터** — 기존 마크다운 에디터(툴바·미리보기)를 큰 높이로 붙여 글쓰기 편의 개선.
3. **성적 평가 표시** — A/B/C/D 그레이드(승인된 평가지표 개수 기반) + **같은 그룹 내** 자산가치 상위 N%.

## 왜 필요했나요?

- 사업계획서는 외부 링크보다 파일로 제출하는 경우가 많고, 다른 학생에게 보이면 안 되는 민감한 내용입니다 → 비공개 첨부가 필요했습니다.
- 회고 에세이는 길어서 일반 textarea 로는 작성이 불편했습니다 → 에디터 + 충분한 높이.
- 학생이 자신의 위치를 가늠할 수 있도록 그레이드와 (같은 등급 안에서의) 자산 순위를 보여줍니다.

## 어떻게 만들었나요?

### 비공개 파일 (핵심: 진짜로 비공개)
- 일반 업로드(`/uploads`)는 정적 서빙이라 URL 만 알면 누구나 접근 가능합니다. 그래서 첨부 파일은 **정적 서빙되지 않는 별도 디렉토리(`/data/private`)** 에 저장하고, **인증 다운로드 엔드포인트**로만 접근하게 했습니다.
- `GET /api/milestones/files/:id` 가 요청자가 **업로더 본인 또는 관리자**인지 검사한 뒤에만 파일을 내려줍니다(아니면 403). PDF·이미지는 브라우저에서 바로 열람(inline), 그 외는 다운로드됩니다.
- 새 테이블 `milestone_files`, 엔드포인트 `POST/GET/DELETE /api/milestones/files`. 배포 시 첨부가 사라지지 않도록 blue/green 공유 영구 볼륨(`prod_private`)을 추가했습니다.

### 성적 / 자산 percentile
- 그레이드 = 승인된 평가지표 개수(4→A, 3→B, 2→C, 1→D, 0→미진입).
- 같은 그레이드 그룹 학생들끼리만 총자산(현금+주식+회사지분−부채)을 비교해 "상위 N%"를 계산합니다. 다른 그룹은 비교에서 제외(격리)됩니다.

### 회고 에디터
- 피드·댓글에서 쓰던 `MarkdownEditor` 를 회고 본문에 재사용(높이 크게). 내용은 마크다운 텍스트라 기존 AI 작성 확률 평가와 그대로 호환됩니다.

## TDD
- 파일 접근제어: 본인 다운로드 OK / 타인 403 / 관리자 OK / 본인 삭제 / 타인 삭제 거부 / 비허용 확장자·타입 거부.
- 성적·percentile: 같은 그룹 내 순위 정확도 + **다른 그룹 학생이 percentile 계산에서 제외되는지**(그룹 격리) 검증.

## 배운 점
- **"비공개"는 URL 숨김이 아니라 접근 검증**: 정적 서빙되는 폴더에 두면 UUID 를 알면 새어나갑니다. 별도 폴더 + 인증 엔드포인트라야 진짜 비공개입니다.
- **영구 볼륨을 잊지 말 것**: blue-green 무중단 배포는 컨테이너를 갈아끼우므로, 첨부를 컨테이너 로컬에 두면 배포마다 증발합니다.
