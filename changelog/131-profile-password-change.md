# 131. 프로필 페이지 비밀번호 변경

**날짜**: 2026-06-12
**티켓**: #131
**타입**: 신규 기능

## 무엇을 했나요?

프로필 페이지에 **비밀번호 변경** 메뉴를 추가했습니다. 현재 비밀번호를 확인한 뒤 새 비밀번호(8자 이상)로 바꿀 수 있습니다.

## 왜 필요했나요?

#128에서 만든 "비밀번호 찾기"는 로그인하지 못하는 사용자를 위한 기능입니다. 로그인한 상태에서 비밀번호를 바꾸고 싶을 때(예: 보안상 주기적 변경)는 방법이 없었습니다.

## 어떻게 만들었나요?

- **Backend**: `PUT /api/auth/password` (JWT 필요) — 현재 비밀번호를 bcrypt로 대조한 뒤에만 변경을 허용합니다. 토큰을 탈취당해도 현재 비밀번호를 모르면 바꿀 수 없습니다.
- **Frontend**: 프로필 페이지 메뉴에 "비밀번호 변경" 항목 → 다이얼로그(현재/새/확인 3필드).
- **TDD**: 테스트 4건 먼저 작성 — 현재 비번 오류·약한 비번·미인증 거부, 성공 시 구 비번 무효화 + 새 비번 로그인.

## 배운 점

- **비밀번호 변경에는 항상 현재 비밀번호 재확인**: 세션(JWT)만으로 변경을 허용하면, 자리를 비운 사이 누군가 기기를 만지는 것만으로 계정을 뺏길 수 있습니다.
