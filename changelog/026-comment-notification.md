# 026. 댓글 알림

**날짜**: 2026-03-23
**태그**: `알림`, `댓글`, `푸시`, `이메일`

## 무엇을 했나요?

내 게시물에 다른 사용자가 댓글을 달면 푸시 + 이메일 알림이 가도록 구현했습니다.

## 왜 필요했나요?

게시물에 댓글이 달려도 알 수 없어서, 피드를 직접 확인하지 않으면 대화를 놓치는 문제가 있었습니다.

## 어떻게 만들었나요?

- `NotifNewComment` 알림 타입 추가 (푸시/이메일 발송 대상)
- `CreateComment`에서 게시물 작성자 ≠ 댓글 작성자일 때만 알림 발송
- 알림 본문에 댓글 미리보기 (50자)
- 프론트엔드 알림 아이콘 매핑 추가

## 사용한 프롬프트

```
내가 포스팅한 글에 댓글이 달리면 알림이 가도록 해줘. 이메일 알림 포함.
```

## 배운 점

- 알림은 "본인 → 본인" 케이스를 반드시 제외해야 합니다. 안 그러면 자기 댓글에 자기가 알림 받게 됩니다.
