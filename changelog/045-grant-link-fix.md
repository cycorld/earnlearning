# 045. 정부과제 바로가기 링크 수정

> **날짜**: 2026-04-10
> **태그**: `fix`, `라우팅`, `정부과제`

## 무엇을 했나요?

공지 게시글의 "지원하러 가기" 링크가 `/grants/13` (복수형) 으로 생성됐는데
프론트엔드 React Router 는 `/grant/:id` (단수형) 만 등록돼 있어서 클릭하면
빈 화면이 뜨던 버그 수정.

## 수정

1. **백엔드**: `grant_usecase.go` 의 auto-post 링크를 `/grant/%d` 로 수정
2. **프론트엔드**: `/grants/:id` → `/grant/:id` 리다이렉트 Route 추가
   (기존에 이미 게시된 공지의 옛 링크도 동작하도록)
