# 033. 프로필 상세 활동 내역 + 아바타 변경

> **날짜**: 2026-04-02
> **태그**: `프로필`, `UX`, `아바타`

## 무엇을 했나요?

1. 유저 프로필 페이지에 게시글, 마켓 포스팅, 정부과제 지원 내역 섹션을 추가했습니다.
2. 내 프로필에서 아바타 클릭 시 사진 변경/삭제가 가능하도록 했습니다.

## 어떻게 만들었나요?

### 활동 내역
- `GET /users/:id/activity` API 추가 — 포스트(최근 20개), 프리랜서 잡, 과제 지원을 한번에 조회
- UserProfilePage에 게시글/마켓/정부과제 3개 섹션 추가, 각 항목 클릭 시 상세 페이지로 이동

### 아바타 변경
- `PUT /auth/avatar` API 추가 — avatar_url 업데이트 (빈 문자열이면 기본 아바타로 초기화)
- 프로필 아바타에 hover 시 카메라 오버레이 표시
- 클릭 시 DropdownMenu: "사진 변경" (파일 업로드) / "사진 삭제" (기본 아바타로)

## 배운 점

- 여러 도메인의 데이터를 한 API로 묶으면 프론트엔드에서 여러 번 호출하지 않아도 됩니다.
- 아바타처럼 간단한 필드 업데이트도 별도 엔드포인트로 분리하면 범용적으로 사용 가능합니다.
