## 설명
2주에 한 번 정기적으로 재산세를 징수한다.

## 규칙
- 은행 잔고(지갑 balance)에서 2,000만원을 공제한 초과 금액에 대해 15% 징수
- 예: 잔고 5,000만원 → (5,000만 - 2,000만) × 15% = 450만원 징수
- 잔고 2,000만원 이하면 징수 없음

## 작업 내용
- Backend: 재산세 계산 + 징수 로직 (cron 또는 관리자 수동 실행)
- 징수 시 transaction 기록 (tx_type: "property_tax")
- 징수 전 학생들에게 사전 알림 (1일 전 공지)
- 징수 후 개별 알림 (징수 금액 포함)
- Admin 페이지에서 수동 징수 버튼 + 다음 징수일 표시
- 프론트: 지갑 페이지에 다음 징수일 + 예상 세금 표시
